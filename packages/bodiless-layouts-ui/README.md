# `@bodiless/layouts-ui`

Default UI for bodiless layouts components.

## Usage

To use a styled version of the `FlexboxGrid` component from `@bodiless/layouts`,
import from this package instead, eg:

```js
import { FlexboxGrid } from '@bodiless/layouts-ui';

export default () => (
  <FlexboxGrid
    ...
)
```
