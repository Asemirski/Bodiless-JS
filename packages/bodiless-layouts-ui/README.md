# `@asemirsk/layouts-ui`

Default UI for bodiless layouts components.

## Usage

To use a styled version of the `FlowContainer` component from `@asemirsk/layouts`,
import from this package instead, eg:

```js
import { FlowContainer } from '@asemirsk/layouts-ui';

export default () => (
  <FlowContainer
    ...
)
```
